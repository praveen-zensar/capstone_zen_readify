# Redis Installation and Service Start

`brew update`

`brew install redis`

`brew services start redis`


# MongoDB data SEED for Few Books 

`yarn seed`

if you didn't install the Mongo Follow Below commands 

`brew install mongosh`

`mongosh --version`
   
   OR use Below link to download 

   `https://www.mongodb.com/docs/mongodb-shell/install/`
   
# All Service Started 

`yarn start:all`

# Run GateWay Services

`yarn start:gateway`
